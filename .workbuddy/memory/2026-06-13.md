# 2026-06-13 — NeuraGEM Beukers Task Replication

## Completed: NeuraGEM Figure 6 replication (Beukers et al. sequence learning task)

**Setup**: Ran NeuraGEM on the two-Markov-chain sequence learning task from Beukers et al. (2024), matching the paper's Figure 6 setup.

**Key config**: Python interpreter = system miniforge3 (/Users/qlu/miniforge3/bin/python3); managed python has code-signing issues with native libs.

**Critical fix discovered**: The `seq_learnConfig` default uses `seq_len=18`, but the paper uses `seq_len=6`. Results were inverted (interleaved > blocked) with seq_len=18 because the LSTM could distinguish contexts from pattern alone. seq_len=6 (matching the paper) produces the expected curriculum effects.

**Results (seed=42, NeuraGEM model)**:

| Curriculum           | Tcid Accuracy | Z Tcid β | Z Trnd β | Paper Match |
|---------------------|---------------|----------|----------|-------------|
| Blocked             | 1.000 (16/16) | 0.503    | 0.537    | ✅           |
| Interleaved         | 0.562 (9/16)  | 0.305    | 0.007    | ✅ (impaired) |
| Interleaved→Blocked | 1.000 (16/16) | 0.647    | 0.399    | ✅ (recovery) |

**Key behavioral signatures reproduced**:
1. Blocked training → high accuracy (Z tracks true context, ctx 0 Z=[0.89,-0.89], ctx 1 Z=[-0.27,0.27])
2. Interleaved training → impaired performance (Z barely differentiates contexts: ctx 0 Z=[0.12,-0.12], ctx 1 Z=[0.13,-0.13]; model biased toward predicting state 5)
3. Mixed curriculum → recovery for this seed (would need ~20 seeds for bimodal distribution)

**Z dynamics confirm mechanistic story**:
- Blocked: Z std=0.922, range=4.26 (Z actively tracks context during testing)
- Interleaved: Z std=0.194, range=0.90 (Z barely moves — gradient w.r.t. Z is near-zero)
- Interleaved_blocked: Z std=0.914, range=4.33 (Z recovers active tracking)
→ The interleaved-trained LSTM learned to ignore Z (Z provided no useful signal during rapid context switching), hence Z can't help even with faster LR

**No suspicious findings**: All test-phase configs verified identical; accuracy computation correct (uses preds[idx+1] with predict_first_frame=True)

## EGO Beukers Task Sweep (13 Jun)

Task conditions equated to NeuraGEM: phase lengths (2500/2500/700+2500), low-level latent='shuffle'.
